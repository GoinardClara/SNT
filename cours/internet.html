Chapitre 1 : Internet et le Web

1. Que désigne "Le Web" ?

Le Web désigne le World Wide Web, un système qui permet de consulter des pages de sites via des navigateurs web à l'aide d'Internet.

2. Quelle est la condition essentielle pour votre ordinateur pour accéder au Web ?

Il doit être connecté à Internet.

3. Où sont stockées toutes les ressources du Web ?

Les ressources du Web sont stockées sur des serveurs, des ordinateurs puissants connectés en permanence à Internet.

4. Expliquer le processus des échanges client-serveur

Le client (navigateur) envoie une requête au serveur pour demander une ressource (ex : une page). Le serveur répond avec le contenu demandé (HTML, image, etc.).

5. Nommer des protocoles qui régissent les échanges entre le client et le serveur

HTTP (HyperText Transfer Protocol) et HTTPS (version sécurisée).

6. Qu'est-ce qu'une URL ? De quoi est-elle composée ?

Une URL (Uniform Resource Locator) est l'adresse d'une ressource sur le Web.
Exemple : https://www.exemple.com/index.html
Elle est composée :

  • du protocole (https)
  • du nom de domaine (www.exemple.com)
  • du chemin d'accès au fichier (/index.html)

7. Qu'est-ce qu'un moteur de recherche ?

Un moteur de recherche est un outil permettant de trouver des pages web à partir de mots-clés. Exemple : Google, Bing, Qwant.

8. Expliquer l’usage et le fonctionnement des robots Crawler ou Spider

Ce sont des programmes qui parcourent automatiquement le Web pour explorer et indexer les pages pour les moteurs de recherche.

9. Comment un moteur de recherche propose des URL à partir de mots-clés ?

Il compare les mots-clés à son index de pages, puis classe les résultats en fonction de leur pertinence.

10. Qu’est-ce qu’une page web ? Que peut-elle contenir ? Nommer un langage pour en créer une.

Une page web est un document consultable sur le Web. Elle peut contenir du texte, des images, des vidéos, des liens... Elle est créée avec le langage HTML.

11. Comment appelle-t-on le système utilisé par le langage HTML ?

Le système de balises (ou tags).

12. Qu’est-ce que le CSS ?

Le CSS (Cascading Style Sheets) est un langage qui permet de styliser les pages HTML (couleurs, mise en page, tailles, etc.).

13. Est-ce que la réutilisation d’une page Web est toujours autorisée ? Nommer un modèle de licence.

Non, cela dépend des droits d’auteur. Certaines pages sont sous licence Creative Commons, qui peut permettre la réutilisation sous conditions.
